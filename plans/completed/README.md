# Completed plans

Finished work lands here — one file per plan, moved out of [../todo.md](../todo.md)
or [../priority-tasks.md](../priority-tasks.md) once it's actually done.

The point is to keep the active lists short without throwing away the reasoning.
When a plan moves here, leave the original notes intact and add a line at the top
saying when it shipped and anything that turned out differently than planned.

Name files after the work, not the date: `dual-use-index.md`, `scan-image-archive.md`.
