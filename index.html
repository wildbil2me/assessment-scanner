⚙ Bridge connection

One Apps Script deployment serves every class — currently on this class

Setup (once): script.google.com → New project → paste Code.gs from this folder → Run setup to authorize → Deploy → New deployment → Web app → Execute as User accessing the web app → copy the /exec URL here. Your class list lives in your own Google account, and each class gets its own Sheet.

📷 Save scanned sheets to Drive (optional)

Keep a photo of each scanned sheet so you can review a paper and override an ambiguous read

Create an OAuth Web application client at console.cloud.google.com → APIs & Services → Credentials, add this page's address under Authorized JavaScript origins, and paste the Client ID here. Images upload to Drive/Quiz Sheets/<class>/<quiz> using the drive.file scope, so this app only ever sees files it created — a grant kept separate from the bridge's. Leave blank to keep scanned images on this device only.

⚙ Manage Classes

Active Classes
Creating a class makes a new Google Sheet in your Drive whose first tab is student-info (the roster). Every quiz you create becomes another tab in that same Sheet. Archiving hides a class without touching its Sheet; deleting an archived class removes it from Quiz Sheets only — the Sheet stays in your Drive and can be re-linked with “Add Existing Sheet”.
